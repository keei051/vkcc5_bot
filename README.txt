Как запустить бота на Mac:

1. Откройте Terminal.
2. Перейдите в папку проекта:
   cd ~/Desktop/vkcc_mac_full_ready
3. Создайте виртуальное окружение:
   python3 -m venv venv
4. Активируйте его:
   source venv/bin/activate
5. Установите зависимости:
   pip install -r requirements.txt
6. Запустите бота:
   python main.py
